# 环境变量

注意

以下环境变量仅供调试使用

# SURGIO_NETWORK_TIMEOUT

  • 默认值: 20000

# SURGIO_NETWORK_CONCURRENCY

  • 默认值: 5

# SURGIO_REMOTE_SNIPPET_CACHE_MAXAGE

  • 默认值: 43200000（12 小时）

# SURGIO_PROVIDER_CACHE_MAXAGE

  • 默认值: 600000（10 分钟）
